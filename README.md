# 🚀 Featured Project

## 📈 SmartInvest

SmartInvest is a web-based portfolio optimization application developed during the **Coding Camp powered by DBS Foundation**.

### Features

- 📊 Portfolio Optimization
- 📉 Mean Variance Model Portfolio (MVMP)
- 📈 Single Index Model (SIM)
- 📅 Dynamic Stock Filtering
- 📊 Financial Risk Analysis
- 🌐 Interactive Streamlit Dashboard

### Built With
